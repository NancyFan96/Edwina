# Edwina



This is a 0.3 version for Robot Roman's detecting facial expression.

Using ASM face landmark, relative features to personal neutral face and SVM.



#### Requirements 

- Python 2.7.14
- dlib (19.7.0)
- face-recognition (1.0.0)
- metric-learn (0.3.0)
- numpy (1.13.3)
- opencv-python (3.3.0.10)
